危険物物流における「混載」——移送と運搬で何が違うのか

法令は改正があり得るため、最終判断は原文（消防法・危険物の規制に関する政令/規則）および所轄消防への確認が前提です。

1. 導入：このテーマがクライアント業務のどこに効くか

混載の理解は、配車コスト・積載効率・納期・事故リスクのすべてに直結します。

「載せていいのか、ダメなのか」の判断を間違えると、最悪は法令違反＋事故、軽くても積載効率の大幅低下や不要な車両手配（＝コスト増）が発生します。逆に、混載ルールを正確に使いこなせば「1台で済むところを2台出していた」ムダを潰せます。

コンサル・営業職にとっては、荷主への提案や物流会社との会話で「混載できるか／できないか」の判断軸を持っていること自体が信頼の入口になります。

2. 基本定義：まず言葉を揃える

混載とは何か

混載＝1つの輸送単位（車両・タンク・コンテナ等）に、性質の異なる複数の危険物を同時に積むこと。

「性質の異なる」がポイントです。同じ第4類・第2石油類のA製品とB製品を一緒に積むのは、法令上の「混載」問題にはなりにくい（ただし化学的相性＝コンタミの問題は別途ある）。法令が気にしているのは、「類」が異なる危険物を一緒にしたとき、互いの危険性が増幅しないか」という点です。

移送と運搬の違い（前提の復習）

移送 運搬
定義 移動タンク貯蔵所（タンクローリー）で危険物を運ぶこと 車両に容器を積載して危険物を運ぶこと
法的性格 「貯蔵所が動いている」扱い → 施設規制に近い 「容器を積んだ車が走っている」扱い → 運搬規制
イメージ タンクと車体が一体。ハード（設備）ごと移動 ドラム缶・IBCなどの容器を荷台に載せて運ぶ

この区分が分かっていないと、混載の話が噛み合いません。移送と運搬では、混載に対する規制の"かかり方"がまったく違うからです。

3. 実務の急所：現場で必ず問題になるポイント

急所① 移送（タンクローリー）には「混載」という概念がほぼ存在しない

ここが最大の誤解ポイントです。

移動タンク貯蔵所はタンク室ごとに危険物を入れる構造ですが、そもそも1つのタンク室に異なる類の危険物を混合して入れることは想定されていません。タンクの構造基準（間仕切り、材質、安全装置）が「単一の危険物を入れる前提」で設計されているからです。

したがって、移送において「混載できるか？」という問い自体が成立しにくい。正確には、複数のタンク室にそれぞれ異なる危険物を入れて走ること自体は、構造基準と取扱基準を満たしていれば可能ですが、それは「混載」というより「区画ごとの貯蔵」に近い発想です。

実務上の制約はむしろ化学的な問題（コンタミ・前荷の残液）やタンク洗浄コストに集中します。

急所② 運搬（容器輸送）の混載には「類の組合せ表」がある

運搬の混載規制は危険物の規制に関する規則 第46条に根拠があります。

考え方はシンプルで、「第○類と第○類は一緒に積んでよい／ダメ」という組合せが決まっているということです。

基本ロジックは以下のとおりです。

  • 酸化性の物質（第1類・第6類）と、可燃性の物質（第2類・第4類・第5類）は原則NG → 酸化剤と燃料を隣に置けば火災・爆発リスクが跳ね上がる
  • 第3類（自然発火性・禁水性）は多くの類との混載がNG → 水や空気と反応する物質は、他の類との接触リスクが制御しにくい
  • 第4類同士（引火性液体どうし）は類が同じなので混載規制の対象外。ただし品名・性状が異なれば実務上の注意は別途必要

覚え方としては、「酸化 × 可燃 = NG」「禁水物は基本孤立」という2原則を押さえれば、表の大半は論理で再現できます。

急所③ 法規OKでも実務NGになる典型パターン

混載が法的に許されていても、以下の理由で現場が止めることがあります。

  • 荷主の契約条件で「専用車」指定：化学メーカーは品質管理上、自社製品以外との混載を認めないケースが多い。これは法規ではなく「契約（＝第四の法規）」
  • コンタミリスク：法令上は混載OKの組合せでも、微量の蒸気接触で製品品質がNGになる場合がある。特に高純度品・食品添加物グレード
  • 事故時の対応が複雑化：漏えいが起きたとき、混載していると「何が漏れたか」「何と何が接触したか」の特定が遅れる → 初動が遅れる → 被害拡大

急所④ 指定数量未満でも混載規制は逃げられない

「量が少ないから混載は自由」と思いがちですが、運搬における混載制限は指定数量の多寡に関係なく適用されます。容器で運搬する以上、類の組合せルールは指定数量未満でも守る必要があります。

ここは初心者が最も間違えやすいポイントの一つです。

急所⑤ 混載の可否判断は「配車の起点」になる

実務的に見ると、混載判断は単なるコンプライアンスチェックではなく配車計画の出発点です。

「A荷主の第4類とB荷主の第1類を1台に載せたい」→ 混載NG → 車両が2台必要 → コスト倍増・ドライバー2名必要 → 納期調整……という連鎖が起きます。だからこそ営業・企画段階で混載の可否を把握しておくことが、見積精度とオペ設計の質を決めるのです。

4. ひと目整理

比較軸 移送（タンクローリー） 運搬（容器積載）
混載の概念 タンク室が物理的に区画されているため、「混載」というより「区画別貯蔵」 荷台上に異なる類の容器が並ぶ → 典型的な「混載」
規制の根拠 構造基準＋取扱基準（移動タンク貯蔵所の技術基準） 危規則 第46条（類の組合せ制限）
判断のキモ 化学的相性・コンタミ・洗浄が主な論点 類の組合せ（法規）＋荷主ルール（契約）が二重にかかる
「法規OK、実務NG」の主因 前荷の残液による品質汚染、洗浄コスト 荷主の専用車指定、コンタミリスク、事故時の複雑化
コストへの影響 洗浄費・回送費・タンク室割り当ての最適化 車両台数・配車効率・積載率に直結

5. クライアント理解への翻訳

現場の制約 → KPIへの影響

  • コスト：混載NGが増えるほど車両台数が増え、配送単価が上がる
  • リードタイム：混載不可による車両手配の遅れが納期を押す
  • 積載効率：混載可能な組合せを最大活用すれば、効率は大幅改善。逆にルールを知らなければ無駄な空車が走る
  • 事故リスク：混載の組合せミスは、漏えい時に被害が指数的に拡大する（酸化剤＋可燃物の接触は最悪シナリオ）

よくある対立：荷主 vs 物流会社

荷主（品質管理部門）は「何があっても混載しないでほしい。専用車で。」と言います。物流会社（配車担当）は「法規上OKな組合せまで全部NGにされると車が回らない。コストも上がる。」と言います。

この対立を調停するには、「法規上の混載可否」と「品質・契約上の混載可否」を分離して見える化し、どこまでが法令のハードルで、どこからが荷主の任意ルールかを整理することが第一歩です。これができると、不要な専用車コストを削る提案にもつながります。

6. すぐ使えるツールキット

ヒアリング質問（5問）

  1. 「混載がNGになっている理由は法規ですか、それとも荷主さんの契約条件ですか？」 → 制約の出所を切り分ける最重要質問
  2. 「現状、車両1台あたりの積載率はどのくらいですか？空きスペースが出ている路線はありますか？」 → 混載活用余地の発見
  3. 「タンクローリーの場合、洗浄の頻度とコストはどの程度かかっていますか？」 → 移送側の「実質的な混載制約」の定量化
  4. 「混載可否の判断は誰が、どの時点で行っていますか？」 → 配車フローのボトルネック発見
  5. 「過去に混載がらみのヒヤリハットや荷主クレームはありましたか？」 → リスクの実態把握

地雷（言ってはいけない／勘違いしやすいこと）3つ

  1. 「少量なら混載しても大丈夫ですよね？」 → 指定数量未満でも混載規制は適用される。量の多寡で免除されない
  2. 「タンクローリーでも混載すればコスト下がりますよね？」 → 移送に「混載」概念を安易に持ち込むと話が噛み合わない。構造・洗浄の文脈で語る必要がある
  3. 「第4類同士なら何でも一緒に積めますよね？」 → 法規上は類が同じなので混載規制の対象外だが、化学的相性やコンタミの問題は別。法規OKで品質NGのパターン

まとめ：覚えるべき3点

  1. 混載は「運搬（容器輸送）」の世界の概念。移送（タンクローリー）では構造上の区画管理とコンタミ管理が論点になる。 両者を混同すると議論がずれる
  2. 運搬の混載可否は「酸化×可燃＝NG」「禁水は孤立」で大枠を掴める。 ただし法規OKでも荷主ルール・品質要件で実務NGになる二重構造がある
  3. 混載判断は配車計画の起点であり、コスト・納期・積載効率に直結する。 「制約の出所（法規か契約か）」を切り分けられることが、提案の精度を決める
