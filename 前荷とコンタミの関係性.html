「前荷（ぜんか）」とコンタミ：洗浄グレードがコストを生む仕組み

法令は改正があり得るため、最終判断は原文・所轄消防への確認が前提です。本稿は業務理解のための教材であり、法的助言ではありません。

1. 導入：このテーマがクライアント業務のどこに効くか

危険物物流のコストを分解すると、「洗浄」と「回送（空車を走らせること）」が想像以上に大きい。その根っこにあるのが 前荷とコンタミ の問題。

これが効くのは以下の局面：

  • コスト：洗浄費・回送費・待機時間が物流原価の10〜20%に達するケースがある
  • 品質：コンタミ（後述）が起きれば製品廃棄＝数百万〜数千万円単位の損害
  • 納期：洗浄待ち・車両不足がリードタイムを直撃する
  • 安全：化学的に相性の悪い残液が混ざれば、発熱・発火・有毒ガス発生のリスク

つまり「前の荷物の残りが、次の荷物を汚す」というシンプルな構造が、コスト・品質・安全のすべてを同時に動かす。クライアントとの会話で「洗浄コストが高い」「車両が足りない」という話が出たら、まずこのテーマを疑うべき。

2. 基本定義：最小限の用語と考え方

前荷（ぜんか）とは

タンクローリーやIBCコンテナなどの容器に「直前に積んでいた荷物」のこと。

タンクローリーは1回使ったら捨てるわけではない。Aという液体を運んだ後、同じタンクにBという液体を積む。このとき「A」が前荷。

コンタミ（コンタミネーション）とは

contamination（汚染）の略。前荷の成分が次の荷物に混入すること。

「混入」と言っても、目に見える量とは限らない。ppm（100万分の1）レベルの微量残留でも、化学品・食品原料・医薬品原料では品質不適合になる。

核心をひと言で言えば：

前荷の「残り」が次荷を汚す。これがコンタミ。防ぐために洗浄する。洗浄にはグレードがあり、グレードが上がるほどコストと時間がかかる。

洗浄グレードとは

タンクの洗浄には段階がある。業界で統一された厳密な定義があるわけではないが、実務上おおむね以下の階層で語られる：

グレード 内容の目安 時間・コスト 典型的な用途
水洗い 水で流すだけ 低 同系統品の連続輸送
洗剤洗浄 中性洗剤やアルカリ洗浄剤で洗う 中 異種品への切替え
溶剤洗浄 有機溶剤で前荷成分を溶かし出す 高 樹脂系・高粘度品の後
蒸気洗浄（スチーム） 高温蒸気で内部を処理 高〜極高 高純度品・食品原料向け
分析証明付き洗浄 洗浄後にタンク内を拭き取り分析し、残留値が基準以下であることを証明 極高 医薬中間体・電子材料等

ここで重要なのは、洗浄グレードは「次に何を積むか」と「前に何を積んでいたか」の組合せで決まるという点。前荷が水溶性で次荷も水溶性なら水洗いで済むかもしれないが、前荷が高粘度樹脂で次荷が高純度溶剤なら、溶剤洗浄＋分析証明が必要になる。

3. 実務の急所：現場で必ず問題になるポイント

急所①：洗浄グレードは「荷主が決める」──法令だけでは決まらない

消防法は「危険物の混載制限」や「容器の基準」を定めているが、「タンクをどこまで洗え」という洗浄グレードの規定はない。

では誰が決めるか。荷主（化学メーカー等）の品質基準が事実上の法規になる。

これが意味するのは：

  • 法令をクリアしても、荷主の品質基準を満たさなければ受け入れ拒否される
  • 荷主ごとに基準が違うため、「A社向けはOKだがB社向けはNG」という同じ製品でも判断が分かれる事態が起きる
  • 配車担当は「法令＋化学適合性＋荷主ルール」の3層で判断している

ここが典型的な "法的にOKでも実務でNG" のポイント。コンサルや営業が「法令上問題ないですよね？」と言うと、現場は「法令の話じゃないんです」と噛み合わなくなる。

急所②：「同じ品名」でもグレード違いでコンタミ扱いになる

例えば「トルエン」を前荷で運び、次も「トルエン」を積む場合。同じ物質だから洗浄不要に思えるが、工業用グレードと電子材料用グレードでは純度要求が桁違い。工業用トルエンの微量不純物が電子材料用トルエンに混入すれば、それはコンタミ。

つまり 「品名が同じ＝洗浄不要」ではない。これは初心者が最も間違えやすいポイントの一つ。

急所③：洗浄コストは「見えにくい原価」として物流費に埋もれる

洗浄にかかるコストは単純な洗浄料金だけではない。構造的に以下が積み上がる：

  • 洗浄料金そのもの（洗浄場への支払い）
  • 回送コスト（洗浄場までの空車走行の燃料・時間・人件費）
  • 待機時間（洗浄場の混雑による車両の非稼働時間）
  • 分析費用（拭き取り検査・成績書発行）
  • 車両の機会損失（洗浄中は他の仕事に使えない）

これらが「運賃」に含まれて見積もられることが多く、荷主側からは「なぜこんなに運賃が高いのか」が構造的に見えない。逆に言えば、ここを可視化・最適化できるのがコンサルや企画職の腕の見せどころ。

急所④：「専用車」と「兼用車」の判断が配車効率を左右する

コンタミリスクを完全に排除する最も確実な方法は 専用車化（特定の荷物しか積まない車両） だが、稼働率が下がる。一方、兼用車（複数品目を切替えて積む車両）は稼働率が上がるが、洗浄コストと品質リスクが増える。

専用車 兼用車
コンタミリスク 極低 中〜高（洗浄品質に依存）
洗浄コスト ほぼゼロ 高（切替えの都度発生）
車両稼働率 低（空車回送が多い） 高（帰り荷が取りやすい）
品質管理の負荷 低 高（履歴管理・分析が必要）
向いている場面 大口・定期・高純度品 多品種・スポット・汎用品

この「専用 vs 兼用」の判断が、物流コストと品質リスクのトレードオフの核心。荷主の「品質は絶対」と物流会社の「稼働率を上げたい」がぶつかる典型的な対立軸がここにある。

急所⑤：前荷履歴の「何代前」まで遡るか問題

厳格な荷主は「直前の前荷」だけでなく、「その前」「さらにその前」──2代前・3代前の履歴まで確認を求めることがある。特に食品原料や医薬中間体の輸送では、過去に毒性の高い物質を積んだ履歴がある車両自体を拒否するケースもある。

これは配車担当にとって極めて重い制約で、車両の「経歴書」を管理し続ける必要があることを意味する。

4. ひと目整理：コンタミ問題の構造フレーム

コンタミ問題は「3つの変数の組合せ」で理解できる。


┌─────────────────────────────────────────────┐
│         コンタミリスクの決定構造             │
│                                             │
│  ① 前荷は何か（化学的性質・残留しやすさ）  │
│        ×                                   │
│  ② 次荷は何か（純度要求・反応性）          │
│        ×                                   │
│  ③ 洗浄は十分か（方法・検証・証明）        │
│        ＝                                   │
│  【結果】受入OK / 受入NG / 製品事故        │
└─────────────────────────────────────────────┘

この3変数のうち、①と②は「何を運ぶか」で決まるため動かしにくい。物流側がコントロールできるのは③の洗浄だけ。だからこそ洗浄グレードの設計と管理が物流事業者の競争力になる。

5. "クライアント理解"への翻訳

KPIへの影響マッピング

KPI 前荷・コンタミがどう効くか
物流コスト 洗浄費・回送費・分析費が直接原価を押し上げる。専用車化すれば洗浄費は減るが稼働率低下で単価が上がる
リードタイム 洗浄待ち・洗浄場の空き待ちがボトルネック化。繁忙期は洗浄場渋滞で1日以上の遅延も
積載効率 専用車化＝帰り荷が取れない＝実車率（積荷を載せて走る割合）の低下
事故リスク 化学的に不適合な前荷の残留は、発熱・発火・有毒ガスの原因になりうる
品質クレーム コンタミ発覚＝製品廃棄＋出荷停止＋原因調査。金銭的損害だけでなく取引信頼の毀損

よくある対立：荷主 vs 物流事業者

荷主の論理：「うちの製品は高純度。コンタミは絶対に許容できない。専用車を用意して、洗浄証明も毎回出してほしい。でも運賃は下げたい。」

物流事業者の論理：「専用車にすると帰り荷が取れず稼働率が落ちる。洗浄証明を毎回出すには分析コストもかかる。それを運賃に反映させないと採算が合わない。」

この対立は構造的なもので、どちらかが間違っているわけではない。解決の方向性は「洗浄の仕組みを効率化する」「前荷の組合せを最適化して洗浄回数を減らす」「リスクの大小に応じて管理レベルを段階化する」といった設計の話になる。ここに企画・コンサルが介入する余地がある。

6. すぐ使える

ヒアリング質問（5問）

  1. 「車両は専用車運用ですか、兼用車運用ですか？その判断基準は何ですか？」

    2. → コスト構造と品質管理の方針が一発で見える

  1. 「前荷の履歴管理はどこまで遡っていますか？（1代前？3代前？）」

    2. → 荷主の要求水準と管理負荷の実態が分かる

  1. 「洗浄はどこで行っていますか？自社洗浄場ですか、外部委託ですか？」

    2. → 洗浄のリードタイムとコスト構造、品質管理の自由度が見える

  1. 「洗浄グレードの基準は誰が決めていますか？（荷主指定？自社基準？）」

    2. → 「第四の法規」たる荷主ルールの存在と、その交渉余地が分かる

  1. 「コンタミ事故の発生頻度と、直近の事例はありますか？」

    2. → リスクの現実度と、再発防止の成熟度が見える

地雷（言ってはいけない/勘違いしやすい）3つ

  1. 「同じ製品なら洗浄いらないですよね？」

    2. → グレード違い（工業用 vs 高純度品）でコンタミになる。品名だけで判断してはいけない

  1. 「法令上OKなら問題ないですよね？」

    2. → 洗浄グレードは法令ではなく荷主基準で決まる。法令遵守は最低ラインであって品質保証ではない

  1. 「洗浄コストを下げれば物流費は下がる」

    2. → 洗浄を簡略化すればコンタミリスクが上がり、品質事故1件で洗浄費の何十倍もの損害が出る。コスト削減は「洗浄の省略」ではなく「洗浄の最適化」で語るべき

まとめ：覚えるべき3点

  1. 前荷とは「直前に積んでいた荷物」。その残留成分が次の荷物を汚すのがコンタミ。防ぐ手段が洗浄であり、洗浄にはグレードがある
  2. 洗浄グレードは法令ではなく荷主の品質基準で決まる。「法令＋化学適合性＋荷主ルール」の3層で判断するのが危険物物流の配車の現実
  3. 専用車 vs 兼用車、洗浄コスト vs 品質リスクのトレードオフが物流コスト構造の核心。ここを可視化・設計できることがクライアントへの提案価値になる
